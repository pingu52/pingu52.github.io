# About

## 관심 분야 
- **Yocto / BSP**: 레이어 구조, BitBake, 이미지/패키징, 재현 가능한 빌드
- **Linux Driver**: Device Tree, I2C/SPI/UART, 커널 로그 기반 디버깅
- **Bring-up & Debug**: 부팅/콘솔/네트워크/스토리지, 부트로더-커널-루트FS 연결
- **DevOps**: Docker/WSL2 기반 개발환경 표준화, 문서화 자동화
- **Virtual Machine**: QEMU/KVM 기반 Proxmox VE, VMware 등  

## 요즘 하는 것
- STM32MP2 계열 보드로 **BSP/드라이버 실습** (Yocto 빌드, DT 수정, 센서 붙이기)
- “삽질 로그”를 **재현 가능한 체크리스트**로 정리해두는 중

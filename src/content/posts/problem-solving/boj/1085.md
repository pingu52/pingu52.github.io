## 문제

<https://www.acmicpc.net/problem/1085>

한수는 지금 (x, y)에 있다. 직사각형은 각 변이 좌표축에 평행하고, 왼쪽 아래 꼭짓점은 (0, 0), 오른쪽 위 꼭짓점은 (w, h)에 있다. 직사각형의 경계선까지 가는 거리의 최솟값을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 x, y, w, h가 주어진다.

## 출력

첫째 줄에 문제의 정답을 출력한다.

## 제한

- 1 ≤ w, h ≤ 1,000
- 1 ≤ x ≤ w-1
- 1 ≤ y ≤ h-1
- x, y, w, h는 정수

## 풀이

현재 위치 `(x, y)` 에서 직사각형의 네 변까지 거리를 각각 구하면 됩니다.

- 왼쪽 변까지 거리: `x`
- 아래 변까지 거리: `y`
- 오른쪽 변까지 거리: `w - x`
- 위쪽 변까지 거리: `h - y`

이 네 거리 중 가장 작은 값이 직사각형 경계선까지 가는 최솟값입니다.

## 코드

```c
#include <stdio.h>

int min(int a, int b) {
    return a < b ? a : b;
}

int main(void) {
    int x, y, w, h;
    int answer;

    scanf("%d %d %d %d", &x, &y, &w, &h);

    answer = min(min(x, y), min(w - x, h - y));

    printf("%d\n", answer);
    return 0;
}
```

## 문제

<https://www.acmicpc.net/problem/1000>

두 정수 A와 B를 입력받은 뒤, A+B를 출력하는 문제입니다.

## 입력

첫째 줄에 A와 B가 주어집니다.

## 출력

첫째 줄에 A+B를 출력합니다.

## 풀이

두 정수를 입력받아 더한 뒤 출력하면 됩니다.

## 코드

```c
#include <stdio.h>

int main() {
  int a, b;
  scanf("%d %d", &a, &b);
  printf("%d", a + b);
}
```

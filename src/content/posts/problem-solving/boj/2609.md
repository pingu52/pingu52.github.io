## 문제

<https://www.acmicpc.net/problem/2609>  

두 개의 자연수를 입력받아 최대 공약수와 최소 공배수를 출력하는 프로그램을 작성하시오.

## 입력

첫째 줄에는 두 개의 자연수가 주어진다. 이 둘은 10,000이하의 자연수이며 사이에 한 칸의 공백이 주어진다.

## 출력

첫째 줄에는 입력으로 주어진 두 수의 최대공약수를, 둘째 줄에는 입력으로 주어진 두 수의 최소 공배수를 출력한다.

## 풀이

최대공약수는 **유클리드 호제법**으로 구할 수 있습니다.

두 수 `a`, `b`에 대해

$$
\gcd(a, b) = \gcd(b, a \bmod b)
$$

를 반복하면, 나머지가 0이 되는 순간의 값이 최대공약수입니다.

최소공배수는 최대공약수를 이용해

$$
\operatorname{lcm}(a, b) = \frac{a \cdot b}{\gcd(a, b)}
$$

로 구할 수 있습니다.

## 코드

```c
#include <stdio.h>

int gcd(int a, int b) {
    while (b != 0) {
        int temp = a % b;
        a = b;
        b = temp;
    }
    return a;
}

int main() {
    int a, b;
    scanf("%d %d", &a, &b);

    int g = gcd(a, b);
    int l = a * b / g;

    printf("%d\n", g);
    printf("%d\n", l);

    return 0;
}
```

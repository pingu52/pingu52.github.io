## 문제

<https://www.acmicpc.net/problem/1001>

두 정수 A와 B를 입력받은 다음, A-B를 출력하는 프로그램을 작성하시오.

## 입력

첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

## 출력

첫째 줄에 A-B를 출력한다.

## 풀이

두 정수를 입력받아 뺀 뒤 출력하면 됩니다.

## 코드

```c
#include <stdio.h>

int main() {
  int a, b;
  scanf("%d %d", &a, &b);
  printf("%d", a - b);
}
```

## 문제

<https://www.acmicpc.net/problem/1181>

알파벳 소문자로 이루어진 N개의 단어가 들어오면 아래와 같은 조건에 따라 정렬하는 프로그램을 작성하시오.

1. 길이가 짧은 것부터
2. 길이가 같으면 사전 순으로

단, 중복된 단어는 하나만 남기고 제거해야 한다.

## 입력

첫째 줄에 단어의 개수 N이 주어진다. (1 ≤ N ≤ 20,000)  
둘째 줄부터 N개의 줄에 걸쳐 알파벳 소문자로 이루어진 단어가 한 줄에 하나씩 주어진다.  
주어지는 문자열의 길이는 50을 넘지 않는다.

## 출력

조건에 따라 정렬하여 단어들을 출력한다.

## 풀이

단어들을 모두 배열에 저장한 뒤 정렬하면 됩니다.

정렬 기준은 다음과 같습니다.

- 길이가 짧은 단어가 먼저
- 길이가 같으면 사전 순으로 앞서는 단어가 먼저

정렬이 끝난 뒤에는 같은 단어가 연속해서 나타나므로, 바로 앞 단어와 비교해서 다른 경우에만 출력하면 중복을 제거할 수 있습니다.

## 코드

```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

char words[20000][51];

int compare(const void *a, const void *b) {
    const char *s1 = (const char *)a;
    const char *s2 = (const char *)b;

    int len1 = strlen(s1);
    int len2 = strlen(s2);

    if (len1 == len2) {
        return strcmp(s1, s2);
    }
    return len1 - len2;
}

int main() {
    int n;
    scanf("%d", &n);

    for (int i = 0; i < n; i++) {
        scanf("%s", words[i]);
    }

    qsort(words, n, sizeof(words[0]), compare);

    printf("%s\n", words[0]);
    for (int i = 1; i < n; i++) {
        if (strcmp(words[i], words[i - 1]) != 0) {
            printf("%s\n", words[i]);
        }
    }

    return 0;
}
```
